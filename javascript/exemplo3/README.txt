Preenchimento um SELECT a partir de outro SELECT

Um exemplo clássico do preenchimento de um SELECT a partir de outro é a escolha das cidades de um determinado estado. 

Quando o usuário seleciona o estado toda a página deve ser recarregada, inclusive os valores dos demais campos já preenchidos, para que o SELECT com as cidades seja exibido.

Com AJAX esta tarefa fica bem mais fácil pois basta que seja feita uma consulta das cidades do estado que esta no primeiro SELECT e inserir as cidades no segundo SELECT.
